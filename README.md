# md-html

### Demo地址:
[https://wwk321.github.io/demo/md-html/md-html.html](https://wwk321.github.io/demo/md-html/md-html.html)

> 个人博客上使用的md转html,单独拿出来传到github上来,说是md转html其实不准确,在编写的过程中按自己的喜好稍微修改了一点markdown语法,不过总的来说功能很不完善.

**目前实现的功能:**

* `#`------------标题;
* `>`------------引用;
* `*`------------无序列表;
* 四个空格--------代码块;
* `*`------------斜体;(目前只支持`*`)
* `**`-----------粗体;(目前只支持`*`)
* `--------------单个代码;
* `[]()`---------链接;(目前只支持这一种形式)
